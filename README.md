# FEG Stage PRO 3.5.3

PWA-приложение для расчёта сцен, ферменных конструкций, клиентов, PDF/КП и облачного хранения через Supabase.

## Новое в 3.5.1–3.5.3

- Добавлен отдельный блок **«Блочный конструктор ферм»** внутри страницы ферм.
- Библиотека элементов: фермы 3 / 2.5 / 2 / 1.5 / 1 / 0.5 м, угловые блоки МДМТ, база / блин, аутригер.
- Постановка блоков на сетку, поворот горизонтально / вертикально, режимы «ставить» и «удалять».
- Автошаблоны: портал и рама.
- Расчёт прямого метража, узлов, углов, кубов, автоматических бабышек по правилу 4 шт на каждое торцевое соединение, веса, проката, монтажа, транспорта и итога.
- Отдельная ведомость блоков с количеством, метражом, весом и прокатом.
- Быстрый 2D-режим рисования ферм сохранён как черновой слой. Блочный конструктор работает отдельно как слой комплектации.
- В блочный конструктор встроены паспортные таблицы нагрузок **MDM T29 вид Q** и **MDM T39 вид Q**.
- Добавлен выбор типа фермы T29Q/T39Q, автоматический расчёт пролёта по самой длинной прямой линии, ручной ввод пролёта, проверка равномерной и точечной нагрузки.
- Добавлена таблица консолей Lк для видов T/Q с расчётом допустимых P1/P2 и уравновешивающей нагрузки в точке A.
- Длины между табличными значениями считаются безопасно — по ближайшему большему пролёту, без интерполяции.

## v3.5.8 port geometry

- В расчёте габаритов и авто-пролёта учтён реальный вынос подключённых угловых/U-блоков: **+0.5 м на каждый узел, подключённый к прямой линии фермы**.

- Шаблоны «Портал» и «Рама» теперь трактуют введённые ширину/высоту как внешний расчётный габарит: прямые фермы автоматически становятся короче на вынос углов.
